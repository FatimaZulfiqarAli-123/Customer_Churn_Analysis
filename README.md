# Customer Churn Analysis Project

## Description
This project predicts customer churn for a telecom company using machine learning models:
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## Dataset
Dataset file: `Customer_Churn_Analysis.csv`

## Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Open `Customer_Churn_Analysis.ipynb` in Jupyter Notebook or VS Code
3. Run all cells
